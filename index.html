Usage
  1. 引入高德地图JavaScript文件
    <script src="http://webapi.amap.com/maps?v=1.3&key=你的ak&plugin=AMap.MouseTool,AMap.PolyEditor"></script>
  2. 引入封装的tustMap.js
    <script src="js/tustMap.js"></script>
  3. 若想去掉高德版权LOGO，添加以下样式
    <style>
        .amap-logo, .amap-copyright {
            display:none
        };
    </style>
初始化地图实例
  1. 定义一个地图容器，需指定其id和长宽
    <div id="allmap" style="width: 80%; height: 450px"></div>
  2. 声明配置对象
    var options = {
        id: 'allmap',
        center: [117.278097,39.077101],
        zoom: 15
    };
    需指定地图容器的id，并设置初始经纬度以及初始缩放比，默认为10
  3. 实例化地图对象
    var myMap = new tustMap(mapOptions);;
