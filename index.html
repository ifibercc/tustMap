Usage
  1. 引入百度地图JS
    <script src="http://api.map.baidu.com/api?v=2.0&ak=你的ak"></script>
  2. 引入鼠标绘制工具，分为JS和CSS
    <script type="text/javascript" src="http://api.map.baidu.com/library/DrawingManager/1.4/src/DrawingManager_min.js"></script>
    <link rel="stylesheet" href="http://api.map.baidu.com/library/DrawingManager/1.4/src/DrawingManager_min.css" />
  3. 引入封装的dwzMap.js
    <script src="js/dwzMap.js"></script>
  4. 若想去掉百度版权LOGO，添加以下样式
    <style>
        .anchorBL {
            display:none
        };
    </style>
创建地图Object
  1. 定义一个地图容器，需指定其id和长宽
    <div id="allmap" style="width: 80%; height: 450px"></div>
  2. 声明配置对象
    var options = {
        id: 'allmap',
        initLng: 117.278097,
        initLat: 39.077101,
        initZoom: 11
    };
    需指定地图容器的id，并设置初始经纬度以及初始缩放比，默认为15
  3. 实例化地图对象
    var myMap = new dwzMap();
